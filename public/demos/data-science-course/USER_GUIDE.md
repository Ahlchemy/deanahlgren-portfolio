# Data Science Evolution Course - User Guide

## Welcome to Your Interactive Learning Journey

This comprehensive e-learning course takes you through the fascinating history of data science, from ancient decision-making practices to cutting-edge generative AI. Designed for technically savvy learners who need to understand these concepts quickly and thoroughly.

## Course Overview

**Duration:** 3-4 hours  
**Modules:** 6 interactive modules  
**Learning Style:** Visual, interactive, and practical  
**Color Scheme:** Professional gray and navy design  

### What You'll Learn

1. **Foundation of Data-Driven Decisions** - Ancient to modern decision-making
2. **Two Paradigms of Data Science** - Inferential vs Computational approaches
3. **AI Fundamentals and Relationships** - Understanding AI, ML, DL connections
4. **Evolution Timeline** - Key milestones from 1940s to present
5. **Large Language Models Deep Dive** - Modern generative AI concepts
6. **Real-World Applications** - Practical implementations and future trends

## Getting Started

### System Requirements

- JavaScript enabled
- Local file access permissions

1. Download and extract the course files

## Troubleshooting

### Common Issues

**Course Won't Load:**
- Ensure JavaScript is enabled
- Check file permissions
- Try a different browser
- Clear browser cache

**Images Not Displaying:**
- Verify all files are extracted
- Check folder structure integrity
- Ensure local file access permissions

**Progress Not Saving:**
- Enable local storage in browser
- Check browser privacy settings
- Ensure cookies are allowed

**Navigation Issues:**
- Refresh the page
- Check browser console for errors
- Ensure all JavaScript files are loaded
