Reinforcement Learning Fundamentals

States

Current situation or configuration of the environment

Example: Robot's joint angles, position, or sensor readings

Actions

Set of possible moves or decisions the agent can make

Example: Moving joints or activating grippers in robotic arm

Rewards

Numerical feedback signals indicating action desirability

Example: Positive reward for successfully grasping an object

Policies

Strategy or rules that guide action selection

Example: Neural network mapping states to actions

Real-World Application

In robot maze navigation: State = current position, Actions = move directions, Rewards = reaching goal, Policy = direction selection strategy
